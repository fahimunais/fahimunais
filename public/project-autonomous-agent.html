Autonomous AI Agent

Multi‑agent system (LangGraph) with Sales, RAG‑QA and Announcer agents; DB tools and PA speaker integration.

Project TypeAgentic AI
StatusIn Development
LangGraph DB Tools RAG Agent PA Speaker

Agent Roles

  • SalesAgent: queries DB (SQL tool), evaluates goals, updates CRM.
  • RAGAgent: answers from KB with citations.
  • AnnouncerAgent: triggers PA speaker when targets met.
  • Critic/Guard: reviews actions; stop/adjust policy.

Agent Graph

[Trigger/User]
   -> Orchestrator (LangGraph)
      |-- SalesAgent --(SQL Tool)--> DB
      |-- RAGAgent ----(Retrieve)--> KB/Chroma
      |-- AnnouncerAgent ----------> PA Speaker
      '-- Critic/Guard -----------> approve/adjust/stop

Challenges & Trade‑offs

  • Accuracy (small LLaMA): route by confidence; retrieval‑first; escalate to larger models selectively.
  • Latency: batch/tool‑prefetch; result caching; non‑blocking announcements.
  • Data: custom dev DB for prototyping; move to prod sources later.

Status & Next

  • Core agent loop running; integrations working in dev.
  • Next: confidence routing; evaluation harness; production data sources.
