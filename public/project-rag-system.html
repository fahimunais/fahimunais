Enterprise RAG System

Multi‑bot platform to build grounded chatbots with per‑bot knowledge bases and simple UI for upload, indexing and chat.

Project TypeKnowledge Management
StatusDeployed
Ollama · Mistral ChromaDB Embeddings: MiniLM family

Platform Overview

Users can create multiple chatbots, each isolated with its own namespace and document set. Upload files, crawl links, or paste text; the system chunks, embeds, and indexes into Chroma. Chat sessions cite sources and restrict answers to grounded facts.

Stack

  • LLM: Ollama (Mistral)
  • Vector DB: Chroma (per‑bot namespaces)
  • Embeddings: MiniLM family
  • Retrieval: Hybrid (BM25 + Vector) + rerank

Ingestion & Retrieval Flow

[Docs/Links/Files] -> [Ingestion] -> [Semantic Chunker]
 -> [Embeddings (MiniLM)] -> [Chroma (namespace per bot)]
User -> [Bot UI] -> [Hybrid Retriever] -> [Rerank] -> [Context Builder]
 -> [Mistral via Ollama] -> [Answer + Citations]

Challenges & Mitigations

  • Multi‑tenant isolation: Chroma namespaces; ACL checks in query path.
  • Freshness & dedupe: incremental ingest; content hashing; source de‑duplication.
  • Small‑model accuracy: hybrid retrieval; rerank; tight prompts; refusal policy.

Status & Outcomes

  • Deployed with per‑team bots; single UI for build‑train‑chat.
  • Grounded answers with citations and easy source management.
